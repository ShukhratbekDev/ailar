# Ailar Monorepo

Bu loyiha **Ailar** platformasining monoreposi hisoblanadi.

## Tuzilishi

- **apps/web**: Asosiy Next.js veb ilovasi (AI Katalog, Yangiliklar, Promptlar).
- **packages/ui**: Umumiy UI komponentalar kutubxonasi.
- **packages/eslint-config**: Umumiy ESLint konfiguratsiyasi.
- **packages/typescript-config**: Umumiy TypeScript konfiguratsiyasi.

## Ishga tushirish

1. Bog'liqliklarni o'rnating:
```bash
npm install
```

2. Ilovani ishga tushiring:
```bash
npm run dev
```

Batafsil ma'lumot uchun `apps/web/README.md` fayliga qarang.
